# React + Vite MIS Template

Management Information System built with React + Vite

## Installation

### Prerequisites
- Node.js (v16 atau lebih tinggi)
- npm atau yarn

### Setup
```bash
# Clone repository
git clone <repository-url>
cd nama-project

# Install dependencies
npm install

# Start development
npm run dev

# Build production
npm run build
```

## Available Scripts
```bash
npm run dev      # Development server
npm run build    # Production build
npm run preview  # Preview build
npm run lint     # Code quality check
```
